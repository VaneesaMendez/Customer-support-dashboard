Customer Support Analytics Dashboard

Project Overview:
Analyzed 734 customer support tickets using Microsoft Excel.

Key Metrics:
- Total Tickets
- Closure Rate
- Average Resolution Time
- Customer Satisfaction Score

Tools Used:
- Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

Business Insights:
- Billing issues represented the largest ticket category.
- Closure rate was 35.6%, indicating a significant backlog.
- Lea demonstrated the fastest average resolution time.
